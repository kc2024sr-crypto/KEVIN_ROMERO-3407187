📅 Semana 6 - Lo que aprendí
Esta semana aprendí a usar bucles en JavaScript para procesar colecciones de datos y generar reportes completos dentro de mi dominio de tienda de productos orgánicos y naturales.
Qué aprendí

for...of → recorre cada elemento de un array de forma limpia, sin necesitar el índice
for clásico → recorre un array usando el índice, útil cuando necesito saber la posición exacta de cada elemento
while → repite un bloque mientras una condición sea verdadera, ideal para búsquedas donde no sé cuántas iteraciones necesito
break → detiene el bucle inmediatamente al encontrar lo que busco
continue → salta la iteración actual y pasa a la siguiente sin salir del bucle
Acumulador → variable que se va sumando en cada vuelta del bucle para obtener un total
Contador → variable que aumenta en cada vuelta para contar cuántos elementos cumplen una condición

Qué hice
🧐 Hice un reporte de inventario para mi tienda orgánica con 8 productos en 4 categorías que lista todos los productos con for...of, cuenta cuántos hay por categoría con contadores anidados, calcula el precio total y promedio del catálogo con un acumulador, encuentra el producto más caro y el más barato, genera un reporte numerado con for clásico indicando si cada producto está sobre o bajo el promedio, busca el primer suplemento disponible con while y break, y filtra productos sobre $25.000 usando continue.
