📅 Semana 2 - Lo que aprendí
Esta semana aprendí a usar variables en JavaScript para guardar y organizar información.
Qué aprendí

const → para guardar valores que no van a cambiar
let → para guardar valores que sí pueden cambiar
camelCase → así se nombran las variables, ejemplo: nombreAnimal
UPPER_SNAKE_CASE → así se nombran las constantes fijas, ejemplo: MAX_CAPACIDAD
typeof → sirve para saber qué tipo de dato tiene una variable
null → representa algo que todavía no tiene valor asignado
Conversiones → se puede cambiar un tipo de dato a otro 


Qué hice
🧐 Hice un script que guarda los datos de mi dominio en variables con los tipos correctos muestra sus tipos y hace al menos una conversión.
Lo que no sabía antes y ahora sí sé
Que no es lo mismo usar const que let, y que ponerle un buen nombre a una variable hace el código mucho más fácil de leer.
