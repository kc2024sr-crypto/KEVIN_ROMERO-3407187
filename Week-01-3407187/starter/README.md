📅 Semana 1 - Lo que aprendí
Esta semana aprendí a usar JavaScript básico en la consola con Node.js.
Qué aprendí

console.log() → sirve para mostrar cosas en la consola
string → es texto, va entre comillas "hola"
number → son números como 45 o 1200
boolean → solo puede ser true o false
comentarios → son notas en el código que empiezan con //


Qué hice 
🧐 Hice un script que muestra una ficha de presentación de mi dominio asignado que fue Tienda de Productos Organicos Y Naturales usando solo console.log y valores directos, sin variables todavía.
Lo que no sabía antes y ahora sí sé
Que JavaScript puede correr directamente en la terminal con Node.js sin necesidad de una página web.
