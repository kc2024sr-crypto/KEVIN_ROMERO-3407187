📅 Semana 5 - Lo que aprendí
Esta semana aprendí a usar condicionales en JavaScript para clasificar, etiquetar y acceder a datos de forma segura dentro de mi dominio de tienda de productos orgánicos y naturales.
Qué aprendí

if / else if / else → permite clasificar un valor en múltiples niveles según condiciones encadenadas
Operador ternario ? : → forma corta de escribir una condición de dos resultados posibles
switch → evalúa una variable contra múltiples casos exactos y asigna una etiqueta a cada uno
?? (nullish coalescing) → retorna el valor de la derecha solo si el de la izquierda es null o undefined
?. (optional chaining) → accede a propiedades de un objeto de forma segura sin romper el script si el objeto es null

Qué hice
🧐 Hice un clasificador de productos para mi tienda orgánica que evalúa el stock disponible en cuatro niveles con if/else if/else, determina si el producto está disponible con un ternario, asigna la categoría del producto con switch, y accede de forma segura al detalle y origen del producto usando ?. y ??, mostrando todo en una ficha de consola con template literals.
